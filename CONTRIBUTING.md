# Contributing to EvoSkillKit

We welcome contributions in four forms:

1. New environment adapters
   - Gemini
   - Claude Code
   - Cursor
   - VS Code agent workflows

2. New task-specific skill templates
   - scientific coding
   - combustion simulation
   - data analysis
   - academic writing
   - literature review
   - plotting

3. Better validation protocols
   - code validation
   - data validation
   - writing validation
   - simulation validation

4. Documentation improvements
   - examples
   - diagrams
   - tutorials
   - translations

## Pull Request Rules

Every PR should explain:

- what problem it solves
- which skill file it changes
- whether it adds a new skill, revises an existing skill, or changes a meta-skill
- what validation was performed
