# Change Log

Every skill update must be logged here.

---

## Change Template

```markdown
## Change YYYY-MM-DD

### Changed Files
List changed repository files.

### Update Type
new skill / skill revision / appendix update / meta skill update / archive / deletion.

### Reason
Why this change was made.

### Evidence
Task trajectory evidence.

### Expected Future Benefit
What future error, delay, or ambiguity this update should reduce.
```
