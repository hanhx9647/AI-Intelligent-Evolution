## Skill Template

```markdown
## Skill ID: SKILL_YYYYMMDD_XXX

### Name
Short descriptive name.

### Domain
Coding / debugging / paper writing / data processing / Cantera / mechanism merging / plotting / literature review / prompt engineering / other.

### Trigger Conditions
Use this skill when...

### Skill Body
Concrete procedural guidance.

### Appendix
Valid rules that the agent previously failed to follow carefully. Do not introduce new rules here.

### Verification Method
How to check whether this skill was correctly applied.

### Failure Modes
Known ways this skill can fail or become misleading.

### Evidence
Task date, task summary, files involved, errors observed, final outcome.

### Credit State
- helpful_count:
- harmful_count:
- neutral_count:
- uncertain_count:

### Status
active / trial / revised / archived / disabled.
```
